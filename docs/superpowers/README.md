# docs/superpowers — historical breadcrumbs

This directory is **historical**. The canonical home for Spec-Driven
Development (SDD) artifacts is
[`openspec/changes/`](../../openspec/changes/).

The plans and specs in `docs/superpowers/plans/` and
`docs/superpowers/specs/` predate the OpenSpec workflow. They are
kept for context (commit history, references from older commits)
but are not actively used. New slices go to `openspec/changes/`.

See [`README.md`](../../README.md) and
[`docs/living-loop-triggers.md`](../living-loop-triggers.md) for
the current documentation.
